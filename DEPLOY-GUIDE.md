# Deploy Your China Sourcing Trip App to GitHub Pages

## Total time: ~5 minutes. No coding needed.

---

## Step 1: Create a GitHub Account (skip if you have one)

1. Go to **github.com**
2. Click **Sign Up**
3. Create your account (free)

---

## Step 2: Create a New Repository

1. Go to **github.com/new**
2. Fill in:
   - **Repository name:** `china-trip` (or whatever you want)
   - **Description:** China Sourcing Trip Companion App
   - **Public** (must be public for free GitHub Pages)
   - Check **"Add a README file"**
3. Click **Create repository**

---

## Step 3: Upload Your Files

1. In your new repo, click **"Add file"** → **"Upload files"**
2. Drag and drop BOTH files from the downloaded zip:
   - `index.html`
   - `manifest.json`
3. Click **"Commit changes"**

---

## Step 4: Enable GitHub Pages

1. Go to your repo's **Settings** tab (gear icon)
2. In the left sidebar, click **"Pages"**
3. Under **"Source"**, select:
   - Branch: **main**
   - Folder: **/ (root)**
4. Click **Save**
5. Wait 1-2 minutes

---

## Step 5: Access Your Live App!

Your app is now live at:

```
https://YOUR-USERNAME.github.io/china-trip/
```

(Replace YOUR-USERNAME with your GitHub username)

---

## Step 6: Add to Your Phone Home Screen

### iPhone:
1. Open the URL in Safari
2. Tap the **Share** button (square with arrow)
3. Tap **"Add to Home Screen"**
4. Name it "CN Trip" → tap **Add**

### Android:
1. Open the URL in Chrome
2. Tap the **3-dot menu** (top right)
3. Tap **"Add to Home screen"**
4. Name it "CN Trip" → tap **Add**

It now looks and works like a native app!

---

## Step 7: Enable AI Features (Optional)

1. Go to **console.anthropic.com**
2. Create an account and get an API key
3. In the app, tap the 🤖 robot button
4. Paste your API key
5. Now you can chat with AI to edit your trip!

**Note:** The key is stored locally on YOUR device only. It never leaves your phone.

---

## How Your Data Works

- **Checklists & budget entries** save to your phone's local storage
- Data persists between visits — close and reopen, it's still there
- **Each device is separate** — your phone and laptop have their own data
- To reset: clear browser data for the site

---

## Updating Your App

To make changes later:
1. Edit `index.html` on GitHub (click the file → pencil icon)
2. Or re-upload a new version
3. Changes go live in ~1 minute

Or just ask me here in Claude to modify anything and I'll give you an updated file!

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Page shows 404 | Wait 2 min. Check Settings → Pages is enabled |
| Fonts look wrong | First load needs internet to download fonts |
| AI chat not working | Check API key starts with `sk-ant-` |
| Data disappeared | Cleared browser cache? Data lives in localStorage |
| Want a custom domain | Settings → Pages → Custom domain (optional) |
